Flask Web Application

A simple web application build on Python language with Flask. Flask is a micro web Python framework used for building web applications.




App Features :
  • The application has proper security measures to prevent unauthorized access.
  • The application has proper database management system.
  • Users of the application can upload files to the system.
  • Uploaded files are stored in the local/server system and 'location-information' of the files are stored in the database.
  • Users of the application can see the list of uploaded file in the home page.
  • Users of the application can delete the uploaded files frome the system.
  • Lastly, users of the application can DOWNLOAD the uploaded files asynchronously.
